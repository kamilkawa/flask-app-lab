# Flask Movie App

Aplikacja webowa do zarządzania bazą filmów napisana we Flasku.

## Wymagania

- Python 3.13+
- Flask

## Instalacja i uruchomienie

1. (Opcjonalnie) Utwórz i aktywuj środowisko wirtualne:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate  # MacOS/Linux
   # .venv\Scripts\activate   # Windows
   ```

2. Zainstaluj zależności:
   ```bash
   pip install -r requirements.txt
   ```

3. Uruchom aplikację:
   ```bash
   python app.py
   ```

4. Aplikacja będzie dostępna pod adresem: http://127.0.0.1:5000

## Funkcjonalności

- Wyświetlanie listy filmów (strona główna)
- Dodawanie nowego filmu
- Usuwanie filmów (poprzez zaznaczenie na liście)
